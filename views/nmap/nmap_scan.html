Nmap Scan Overview

This module will run an nmap scan from the Kvasir server and import the results. To use this module, nmap must be present on the system and found in $PATH. Some options require root and all scanning will be performed using the task scheduler.

Default Scan Options are:

Intense
nmap -T4 -A -v
Intense plus UDP
nmap -sS -sU -T4 -A -v
Intense, all TCP ports
nmap -p 1-65535 -T4 -A -v
Intense, no ping
nmap -T4 -A -v -Pn
Ping scan
nmap -sn
Quick scan
nmap -T4 -F
Quick scan plus
nmap -sV -T4 -O -F --version-light
Slow comprehensive
nmap -sS -sU -T4 -A -v -PE -PP -PS80,443 -PA3389 -PU40125 -PY -g 53 --script "default"

The nmap output option "-oX" is always implied.
