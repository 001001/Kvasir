ShodanHQ Overview

ShodanHQ scans the publicly accessible Internet address ranges probing for open services and gathers information.

IP addresses will count against the total scan credits. As of 11/2013 this exchange rate is 1 credit = 1,000 results. Additional information is available here.

Scan results may be downloaded through the use of the Developer API or manually. API calls may require credits from your ShodanHQ account. Cisco Systems is not responsible for the use of this API and any errors that may deduct credits from your account.

To use the Developer API edit your models/db.py file and enter it in 'settings.shodanhq_apikey'
